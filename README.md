# JavaScript Mini Projects

A collection of beginner-friendly JavaScript projects built to strengthen core programming and web development skills.  
Each project helps practice DOM manipulation and event handling. 

---

## BMI Calculator

A simple web-based BMI calculator that allows users to input their height and weight and instantly calculates their Body Mass Index.

**Features:**
- Responsive and beginner-friendly design  
- Real-time calculation and result display  
- Basic input validation  

---

## Number Guessing Game

An interactive guessing game where the user tries to guess a randomly generated number within a certain range.

**Features:**
- Dynamic feedback messages (too high / too low)  
- Keeps track of number of attempts  
- Encourages logical thinking and user interaction  

---

Built using **HTML**, **CSS**, and **JavaScript**.
